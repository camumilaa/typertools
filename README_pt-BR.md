# TyperTools Global Master v3.0 (Photopea)

Olá! Sou um typer de mangá brasileiro. Como meu PC não roda Photoshop, uso o Photopea e, com a ajuda de IA, criei este plugin para facilitar o typesetting, inspirado no TyperTools do Photoshop. Não é perfeito, mas ajuda muito!

## 💡 Por que este projeto foi criado?
Criei esta ferramenta porque o Photopea não possui um sistema nativo de inserção rápida de texto como o Photoshop. Como typer, eu precisava de algo que agilizasse o processo de colocar centenas de falas em um capítulo sem ter que copiar e colar manualmente cada uma.

## 🚀 Para que serve a ferramenta?
A TyperTools serve para automatizar o processo de **typesetting de mangás e quadrinhos**. Ela permite que você insira as falas do seu script diretamente no Photopea, exatamente onde o mouse está posicionado, usando estilos de fonte pré-salvos e filtros que ignoram as partes inúteis do script (como números de página e SFX).

## 📂 Arquivos do Projeto
*   `index.html`: Interface principal.
*   `style.css`: Design e visual (Modo Escuro).
*   `app.js`: Lógica, atalhos e sistema de comunidade.
*   `fonts.js`: Catálogo de fontes do Photopea.

## ⌨️ Atalhos de Teclado
*   **B + A**: Avança para a próxima linha e gera o texto no Photopea (Fluxo contínuo).
*   **Ctrl + Enter**: Gera o texto da linha atual.
*   **Setas ⬅️ / ➡️**: Navega entre as linhas do script.

## 💡 Funcionalidades
*   **Filtros**: Ignora números e termos como "SFX", "PAG", etc.
*   **Estilos**: Salva fonte e tamanho para uso rápido.
*   **Comunidade**: Aba interna para sugestões e bugs com sistema de likes.
*   **Tradução**: Suporte para Português e Inglês.

*Obs: Ainda estou aprendendo a lidar com a seleção de fontes, desculpe se houver bugs nessa parte!*
