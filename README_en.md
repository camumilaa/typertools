# TyperTools Global Master v3.0 (Photopea)

Hello! I'm a Brazilian manga typer. Since my PC can't run Photoshop, I use Photopea and, with the help of AI, I created this plugin to make typesetting easier, inspired by Photoshop's TyperTools. It's not perfect, but it helps a lot!

## 💡 Why was this project created?
I created this tool because Photopea lacks a native fast text insertion system like Photoshop. As a typer, I needed something to speed up the process of placing hundreds of lines in a chapter without having to manually copy and paste each one.

## 🚀 What is the tool for?
TyperTools is designed to automate the **manga and comic typesetting process**. It allows you to insert lines from your script directly into Photopea, exactly where the mouse is positioned, using pre-saved font styles and filters that ignore useless parts of the script (like page numbers and SFX).

## 📂 Project Files
*   `index.html`: Main interface.
*   `style.css`: Design and visual (Dark Mode).
*   `app.js`: Logic, shortcuts, and community system.
*   `fonts.js`: Photopea font catalog.

## ⌨️ Keyboard Shortcuts
*   **B + A**: Advance to the next line and generate text in Photopea (Continuous flow).
*   **Ctrl + Enter**: Generate text for the current line.
*   **Arrows ⬅️ / ➡️**: Navigate between script lines.

## 💡 Features
*   **Filters**: Ignores numbers and terms like "SFX", "PAGE", etc.
*   **Styles**: Saves font and size for quick use.
*   **Community**: Internal tab for suggestions and bugs with a like system.
*   **Translation**: Support for Portuguese and English.

*Note: I'm still learning how to handle font selection, sorry for any bugs in that part!*
