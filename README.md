# TyperTools Global Master - Photopea Edition

A versão mais completa e internacional da TyperTools. Agora com suporte a múltiplos idiomas, catálogo de fontes expandido e fluxo de trabalho ultra-rápido.

## 🌍 Novidades da Versão Global

- **Sistema de Tradução**: Suporte completo para **Português (BR)** e **Inglês (US)**. Alterne instantaneamente no cabeçalho.
- **Catálogo de Fontes Expandido**: 📚 Inclui as fontes mais populares do Photopea, Google Fonts e fontes específicas para Typesetting/Manga.
- **Fluxo de Trabalho Otimizado**: Use o atalho **B + A** para avançar e gerar texto exatamente onde seu mouse está.
- **Interface Refinada**: Design compacto e elegante, otimizado para o painel lateral do Photopea.

## 🚀 Como Usar

1. **Escolha seu Idioma**: Use o seletor no topo (🇧🇷/🇺🇸).
2. **Cole seu Script**: O texto será filtrado automaticamente (números e marcadores ignorados).
3. **Selecione o Estilo**: Escolha um estilo salvo ou crie um novo (🎨).
4. **Use o Catálogo**: Clique em 📚 para ver as fontes reais do Photopea e selecione com um clique.
5. **Gere com Velocidade**: Coloque o mouse no local desejado e pressione **B + A**.

## ⌨️ Atalhos de Teclado

| Atalho | Ação |
|--------|------|
| **B + A** | Próxima linha + Gerar Texto (Fluxo contínuo) |
| **Ctrl + Enter** | Gerar texto da linha atual |
| **Seta Direita (→)** | Próxima linha |
| **Seta Esquerda (←)** | Linha anterior |
